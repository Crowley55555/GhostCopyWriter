<h1 align="center">Ghostwriter</h1>

<p align="center">
  <b>Ghostwriter</b> — это современный веб-сервис на Django для генерации профессионального, цепляющего и SEO-оптимизированного контента для социальных сетей с автоматическим подбором визуала.<br>
  Проект ориентирован на маркетологов, SMM-специалистов, предпринимателей и всех, кто хочет быстро создавать качественные посты для разных платформ.
  
    Проект находится на финальной стадии разработки.
</p>

<hr/>

<h2>Возможности</h2>
<ul>
  <li>Генерация текстов для соцсетей <b>(VK, Дзен, Telegram, TikTok, Instagram, Facebook, Twitter/X, LinkedIn и др.)</b> с учетом платформенных особенностей.</li>
  <li>Гибкая настройка параметров: <i>тон голоса, цель, эмоциональный окрас, формат, стиль, призыв к действию (CTA), длина поста, использование хэштегов и упоминаний, адаптация под целевую аудиторию</i>.</li>
  <li>Автоматическая генерация релевантного изображения к посту.</li>
  <li>Сохранение и управление своими генерациями и шаблонами.</li>
  <li>Регистрация, авторизация, личный кабинет, редактирование профиля.</li>
  <li>Стена пользователя с историей генераций.</li>
  <li>Простой и интуитивный интерфейс.</li>
</ul>

<h2>Как это работает</h2>
<ol>
  <li><b>Пользователь</b> выбирает тему поста и настраивает параметры генерации.</li>
  <li><b>Система</b> с помощью нескольких "агентов" (черновик, маркетолог, блогер, SEO, критик, защитник) создает оптимальный текст, подбирает визуал, хэштеги, ключевые слова и CTA.</li>
  <li><b>Результат</b> — готовый пост с рекомендациями по визуалу, хэштегами и ключевыми словами, который можно сразу использовать.</li>
</ol>

<h2>Быстрый старт</h2>
<ol>
  <li><b>Клонируйте репозиторий</b>
    <pre><code>git clone https://github.com/yourusername/Ghostwriter.git
cd Ghostwriter</code></pre>
  </li>
  <li><b>Установите зависимости</b>
    <pre><code>pip install -r requirements.txt</code></pre>
  </li>
  <li><b>Проведите миграции</b>
    <pre><code>python manage.py migrate</code></pre>
  </li>
  <li><b>Запустите сервер</b>
    <pre><code>python manage.py runserver</code></pre>
  </li>
  <li><b>Откройте в браузере</b><br>
    Перейдите по адресу: <a href="http://127.0.0.1:8000/">http://127.0.0.1:8000/</a>
  </li>
</ol>

<h3>Основные команды</h3>
<ul>
  <li><code>python manage.py createsuperuser</code> — создать администратора</li>
  <li><code>python manage.py collectstatic</code> — собрать статику (если нужно)</li>
</ul>

<h2>Стек технологий</h2>
<ul>
  <li>Python 3.x</li>
  <li>Django</li>
  <li>HTML/CSS (Bootstrap)</li>
  <li>Внешние API для генерации текста и изображений (GigaChat, Yandex и др.)</li>
</ul>

<h2>Структура проекта</h2>
<ul>
  <li><code>generator/</code> — основное приложение: модели, формы, представления, шаблоны</li>
  <li><code>ghostwriter/</code> — настройки Django</li>
  <li><code>media/</code> — сгенерированные изображения</li>
  <li><code>staticfiles/</code> — статические файлы</li>
</ul>
