# Ghostwriter

**Ghostwriter** — это современный веб-сервис на Django для генерации профессионального, цепляющего и SEO-оптимизированного контента для социальных сетей с автоматическим подбором визуала. Проект ориентирован на маркетологов, SMM-специалистов, предпринимателей и всех, кто хочет быстро создавать качественные посты для разных платформ.

## Возможности

- Генерация текстов для соцсетей (VK, Дзен, Telegram, TikTok, Instagram, Facebook, Twitter/X, LinkedIn и др.) с учетом платформенных особенностей.
- Гибкая настройка параметров: тон голоса, цель, эмоциональный окрас, формат, стиль, призыв к действию (CTA), длина поста, использование хэштегов и упоминаний, адаптация под целевую аудиторию.
- Автоматическая генерация релевантного изображения к посту.
- Сохранение и управление своими генерациями и шаблонами.
- Регистрация, авторизация, личный кабинет, редактирование профиля.
- Стена пользователя с историей генераций.
- Простой и интуитивный интерфейс.

## Как это работает

1. **Пользователь** выбирает тему поста и настраивает параметры генерации.
2. **Система** с помощью нескольких "агентов" (черновик, маркетолог, блогер, SEO, критик, защитник) создает оптимальный текст, подбирает визуал, хэштеги, ключевые слова и CTA.
3. **Результат** — готовый пост с рекомендациями по визуалу, хэштегами и ключевыми словами, который можно сразу использовать.

## Быстрый старт

### 1. Клонируйте репозиторий

```bash
git clone https://github.com/yourusername/Ghostwriter.git
cd Ghostwriter
```

### 2. Установите зависимости

```bash
pip install -r requirements.txt
```

### 3. Проведите миграции

```bash
python manage.py migrate
```

### 4. Запустите сервер

```bash
python manage.py runserver
```

### 5. Откройте в браузере

Перейдите по адресу: [http://127.0.0.1:8000/](http://127.0.0.1:8000/)

## Основные команды

- `python manage.py createsuperuser` — создать администратора
- `python manage.py collectstatic` — собрать статику (если нужно)

## Стек технологий

- Python 3.x
- Django
- HTML/CSS (Bootstrap)
- Внешние API для генерации текста и изображений (GigaChat, Yandex и др.)

## Структура проекта

- `generator/` — основное приложение: модели, формы, представления, шаблоны
- `ghostwriter/` — настройки Django
- `media/` — сгенерированные изображения
- `staticfiles/` — статические файлы

## Переменные окружения

Для работы с внешними API могут потребоваться ключи. Пример:

```
GIGACHAT_API_KEY=...
YANDEX_API_KEY=...
```
